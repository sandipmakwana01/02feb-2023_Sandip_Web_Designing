Question 17 : What is the meaning of the CSS selector?

The class , id or tag selector is targets all same name of class , id or tag in the HTML document.
The color property red and the font-size property 36px.
This means to all same name of class , id or tag in the HTML document will be displayed in red with a font size of 36 pixels.

EXAMPLE

CSS selector is h1, targets the all h1 HTML element.
