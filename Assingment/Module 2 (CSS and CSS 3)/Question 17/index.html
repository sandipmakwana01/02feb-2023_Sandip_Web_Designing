Question 17 : What is the meaning of the CSS selector?

div , h1 , p , class , id etc. element is targets all same name of element in CSS style sheet.
The color property red and the font-size property 36px.
This means to all same name of element in the HTML document will be displayed in red with a font size of 36 pixels.

CSS selector is h1, targets the all h1 HTML element.
