Question 11 : How to center block elements using CSS1?

CSS1, the first version of Cascading Style Sheets, introduced the text-align property for centering block-level elements. However, this property only works for inline-level elements inside block-level elements.

To center a block-level element in CSS1, you can use the margin property with auto values.
