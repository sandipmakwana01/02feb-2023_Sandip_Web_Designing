Question 14 : What is embedded style sheets?

Embedded CSS refers to CSS code that is included within the style tag of an HTML document. This type of CSS is placed in the head section of the HTML file and is
used to style specific elements within the page.

Embedded CSS is used to style the h1 and p tags.
