Question 15 : What are the external style sheets?

External style sheets in HTML are used to separate the presentation of the website's content from the structure of the content. This allows for consistent styling across multiple web pages and makes it easier to maintain the website's design.

To use an external style sheet, you must create a separate file with a .css file extension and reference it in the HTML file using the link element.

the link element specifies the location of the external style sheet using the href attribute. The rel attribute specifies the relationship between the current document and the linked document, and the type attribute specifies the MIME type of the linked document (in this case, text/css).

By using an external style sheet, you can define the styles once and apply them to multiple HTML documents. This makes it easier to maintain a consistent look and feel across your website.
