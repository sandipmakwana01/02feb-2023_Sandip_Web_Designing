Question 19 : What is the rule set?

EXAMPLE

the selector is ans19, which targets all ans19 h1 elements in the HTML document. The declarations enclosed in curly braces set the color property to red and the font-size property to 36px. This means that all ans19 h1 elements in the HTML document will be displayed in red with a font size of 36 pixels.

the CSS code is contained in an external stylesheet file called ans19.css, which is linked to the HTML document using the link element in the head section of the HTML code.
