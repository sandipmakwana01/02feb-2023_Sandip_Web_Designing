Question 9 : Which property controls the image scroll in the background?

The CSS background-attachment property controls whether a background image scrolls with the content or remains fixed in place.

There are two possible values for background-attachment:

scroll: This is the default value, where the background image scrolls along with the content as the user scrolls the page.
fixed: This value sets the background image as fixed so that it remains in the same position relative to the viewport even as the user scrolls the page.

The background-attachment property controls whether the background image scrolls with the rest of the content or stays fixed in place.
the background-attachment property is set to fixed, which means that the background image will stay fixed in place while the rest of the content scrolls.
