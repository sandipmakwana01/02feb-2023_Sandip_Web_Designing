Question 18 : What are the media types allowed by CSS?

screen - used for displaying content on a computer screen

<link rel="stylesheet" media="screen" href="styles.css">

@media screen {}

print - used for printing content on paper

<link rel="stylesheet" media="print" href="print.css">

@media print {}

all - applies to all devices

<link rel="stylesheet" media="all" href="styles.css">

@media all {}

handheld - used for small handheld devices like smartphones

<link rel="stylesheet" media="handheld" href="handheld.css">

@media handheld {}

tv - used for television screens

<link rel="stylesheet" media="tv" href="tv.css">

@media tv {}

speech - used for screen readers and other assistive technologies

<link rel="stylesheet" media="speech" href="speech.css">

@media speech {}

These media types can be specified in the media attribute of the link element when linking to an external stylesheet in HTML, and in the @media rule in CSS when defining styles for specific devices or situations. By using media queries, you can create responsive designs that adapt to the characteristics of the device or medium being used to view your website.
