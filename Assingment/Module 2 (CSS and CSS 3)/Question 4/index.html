Question 4 : Name a few CSS style components?

Color:

CSS allows you to set the color of text and background using properties such as color and background-color.

Typography:

CSS provides a range of typography properties such as font-family, font-size, font-weight, line-height, and text-align, which can be used to style the text on a page.

Layout:

CSS provides several layout options such as float, flexbox, and grid, which can be used to arrange elements on a web page.

Box model:

The box model refers to the sizing and spacing of HTML elements. CSS provides properties such as width, height, margin, and padding, which can be used to control the box model.

Borders:

CSS allows you to add borders to HTML elements using properties such as border-style, border-width, and border-color.

Backgrounds:

CSS provides properties such as background-color and background-image, which can be used to style the background of an HTML element.

Transitions and animations: CSS allows you to add transitions and animations to HTML elements using properties such as transition and animation
