Question 16 : What are the advantages and disadvantages of using external style sheets?

Advantages

Reusability :- Creat a first time external style sheet code this style sheet use to multiple time in website.
Consistency :-Use external style sheets and edit one sheet to apply entire website.

Disadvantages:

Limited flexibility :- External style sheets are use only in HTML elements. can not creat dynamic effects to the page. need to use JavaScript or other scripting languages.
Dependence on file location :- external style sheet by chance is move or delete will be effect the website.
