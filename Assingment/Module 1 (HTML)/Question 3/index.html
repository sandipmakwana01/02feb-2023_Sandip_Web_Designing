Question 3 : What are void element in HTML?

-> Most of the HTML element are surronded by start and end tags to specify the starting and end of the element.
-> There is a special group of elements that only have start tags and does not contain any content within it, these elements are called void element.
-> Void elements doesn't have ending tags and can only have attributes but do not contain any kind of content.
-> These element can have backslash before ending of start tag nut that is completely optional.
-> Example of such elements are "br" , "input" , "img" , etc.
